# total_RNASeq

This makes available the code and databases needed to run the total_rnaseq script described in Grant, A and C.S. Davies 2026, Contamination of widely used databases compromises rRNA based taxonomic assignment of metagenomes and metatranscriptomes BioRxiv in press.

The scripts directory needs to be added to the user's PATH. Trimmomatic, Seqkit, USEARCH 12, SortMeRNA and LCA (from LotuS2) need to be installed on the user's system and the location of the databases used set within the total_rnaseq script.
